Vidur Live

Browser Extension · Manifest V3 · 2026

Vidur is a free, open-source browser extension that visually redesigns any website with AI — leaving the content intact. Describe the look you want and optionally point to a reference site, and Vidur restyles the page on the fly. Designs persist across visits. Works on a bring-your-own-key model (API keys never leave your device). Runs on Chrome, Edge, Brave, and Firefox.

↓ Download Git Repo
How to Use
Install on Chrome, Edge, or Brave
  1. Download and unzip vidur.zip
  2. Open chrome://extensions and enable Developer mode
  3. Click Load unpacked and select the vidur folder
Install on Firefox
  1. Open about:debugging#/runtime/this-firefox
  2. Click Load Temporary Add-on → select manifest.json from the unzipped folder
How to use
  1. Navigate to any website and click the Vidur icon in your toolbar
  2. First time: click Open Settings, add an API key for any supported provider
  3. Type a redesign prompt — e.g. "dark terminal with green monospace text"
  4. Optionally paste a reference site URL to match its visual style
  5. Click Redesign — changes apply instantly and persist across visits
  6. Toggle the switch at any time to enable or disable the design for that site
Supported browsers

Chrome 91+ · Edge 91+ · Brave · Firefox 101+ · Safari not yet supported

Aria Live

Pipecat · Deepgram · Sarvam · Vobiz · Cloud Run · Firestore · 2026

Aria is an AI voice sales agent that calls your phone and roleplays a post-demo-class sales conversation in Hinglish — you play a parent after your child's demo class at Vedantu, and Aria gauges interest, reasons through objections, and works to close the sale. Pick a parent persona, enter your phone, and watch its intent classification, objection plays, and next-step decisions in real time. Pipecat pipeline, Deepgram Nova-3 STT + Sarvam Bulbul v3 TTS, key-rotating LLM pool (Groq → Cerebras → Gemini), Vobiz telephony.

Launch Git Repo
How to Use
Try the live demo
  1. Open talktoaria.web.app on a device with a working phone number you can answer
  2. Pick one of five parent personas (Sharma + Aarav + JEE, Reddy + Pranavi + JEE Crash, Iyer + Diya + NEET, etc.)
  3. Enter your phone in E.164 format (e.g. +9198XXXXXXXX)
  4. Click Call me. Pick up — Aria will greet you in Hindi and start the conversation
  5. Roleplay the parent. Try a soft objection ("PW sasta hai"), a stall ("sochna padega"), or a hard no ("interest nahi") — see how the agent's reasoning panel responds in real time
Rate limit

Each phone number can trigger up to 5 demo calls per hour (per build-cost discipline). The dashboard shows your remaining quota; the call button locks at 0.

What's under the hood

FastAPI + Pipecat on Cloud Run · Firestore for CRM + call logs · Deepgram Nova-3 STT + Sarvam Bulbul v3 TTS · OpenAI-compatible LLM proxy with key rotation across Groq, Cerebras, Gemini · Firebase Hosting rewrites for REST · direct Cloud Run WebSocket for media + reasoning panel. Source on GitHub.

HealThee Active

Firebase, Firestore, Groq, Cerebras · 2026

HealThee is a free AI health tracker — a HealthifyMe Plus alternative, optimised for me and forever free. A full-stack personal health dashboard with daily calorie, macro, and micronutrient tracking, plus AI insights. Includes an adaptive meal plan, water intake and weight logging, exercise tracking and sleep monitoring, with trend analysis, weekly/monthly dashboards, and access controlled by Google Auth. Installable as a PWA.

Launch
How to Use
Getting started
  1. Open HealThee and sign in with your Google account
  2. Complete the quick onboarding — enter your height, weight, and gender
  3. Tell the AI what you want to achieve; it will set personalised weight and exercise goals
  4. Choose which areas to track (nutrition, exercise, wellness, skincare, haircare, trends)
Daily tracking
  1. Fuel — log meals (from your plan, by photo, or manually), water, and supplements
  2. Move — follow your personalised workout, log steps and custom exercises
  3. Well — record weight, body metrics, sleep, and daily delights
  4. Care — check off skincare and haircare routines, or chat with the AI health advisor
  5. Stats — review weekly summaries and 7/30/90-day trend charts
Tips

Tap your profile photo to update goals anytime. Works offline as a PWA — install it to your home screen for quick access.
